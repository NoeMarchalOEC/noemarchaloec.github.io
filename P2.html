Projet 2 : Tag nfc

Voici une présentation de l'un de mes projets favoris le tag nfc

Description du projet : Cette objet permet de créer, des tâches avec une interface utilisateur simple et intuitive. L'objectif est de rendre la gestion des tâches plus facile et efficace pour les utilisateurs.

Technologies utilisées :

  • Tag NFC : l'objet physique en lui même.
  • Pythob : Pour la programmation des tag NFC.
  • un lecteur: Permet de lire l'information disponible depuis l'iphone 6 sur IOS.

Fonctionnalités principales :

  • Création de raccourcis.
  • Connection a un réseau particulier hyper rapide.
  • Suppression des taches répétitive.
  • Interface réactive et adaptée aux mobiles.
