Projet 3 : Application de gestion des tâches

Voici une présentation de l'un de mes projets les plus récents : une application web de gestion des tâches.

Description du projet : Cette application permet de créer, modifier et supprimer des tâches avec une interface utilisateur simple et intuitive. L'objectif est de rendre la gestion des tâches plus facile et efficace pour les utilisateurs.

Technologies utilisées :

  • HTML/CSS : Pour la structure et le style de l'application.
  • JavaScript : Pour la gestion dynamique des tâches.
  • Node.js : Backend pour la gestion des données en temps réel.

Fonctionnalités principales :

  • Création de tâches avec titre, description et date d'échéance.
  • Modification des tâches existantes.
  • Suppression des tâches.
  • Interface réactive et adaptée aux mobiles.
